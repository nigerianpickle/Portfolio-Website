Stock Bud is a command‑line assistant that aggregates and distills the day’s top stock‑market headlines into concise, plain‑English summaries—powered by a combination of web scraping and AI. Built as a personal project, it helps me streamline financial news research so i can stay informed in seconds rather than minutes. It’s a work in progress, but it’s already saved me hours of reading time. I plan to add more features in the future, such as a GUI and a mobile app.

  • Automated News Aggregation: Can crawl multiple sources (CNBC, Bloomberg, MarketWatch).
  • AI Summaries: Uses the OpenAI API to convert raw headlines and snippets into a 3–4 sentence summary, which is meant to be readable for non financial gurus.
  • Insights: Uses previously stated API to highlight what stocks the news is likely to affect and how investors should prepare.

  • BeautifulSoup & Requests: ★★★★☆
