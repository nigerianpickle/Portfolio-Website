Easy Quizzy

Easy Quizzy is a quiz app that allows users to create and take quizzes. The app features a user-friendly interface and supports multiple-choice questions. It was developed as part of a group project for the course "COMP 3350" at the University of Manitoba. My role in the project was to develop the Android app using Java and Android Studio. I was responsible for working on the front and backend of the project. I also had the priviledge of writing extensive unit , integration and system tests on the software.
