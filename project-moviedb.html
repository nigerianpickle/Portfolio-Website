Movie,Database

A normalized SQL database of 900,000+ films, optimized for fast lookups is designed to handle complex queries efficiently, making it suitable for data analysis and reporting. This project was developed as part of the course "COMP 3380" at the University of Manitoba. My role in the project was to design the database schema,clean the data, and implement stored procedures for efficient data retrieval.

Project Data:

  • Total Records: 900,451 movies, 120,000 actors, 25,000 directors
  • Normalized to BCNF with 12 tables (movies, actors, crew, ratings, genres, etc.)
